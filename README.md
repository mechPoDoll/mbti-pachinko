## 📖 소개

이 진단은 일반적인 MBTI 검사를 **파친코 홀**이라는 무대로 옮겨온 비유적 콘텐츠입니다.

자리 고르기, 신기종 출시, 잭팟 직후의 행동, 친구가 돈을 빌려달라고 할 때… 12개의 구체적인 상황 속에서 당신이 어떻게 반응하는지를 통해 **16가지 파친코 플레이어 유형** 중 하나로 진단해드립니다.
